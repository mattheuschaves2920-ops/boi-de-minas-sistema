VERSÃO V3 - COM LEITOR DE CÓDIGO DE BARRAS PELO CELULAR

O que foi incluído:
- Leitor de código de barras usando a câmera do celular
- Cadastro de item com leitura por câmera
- Entrada e saída com leitura por câmera
- Categorias prontas para seleção
- Produção da cozinha
- Desperdício
- Comparativo almoço x janta
- Relatórios CSV

Importante:
- O leitor funciona melhor no celular, pelo navegador, com HTTPS
- No Render, o endereço já é HTTPS
- Quando clicar em 'Ler código de barras', o navegador pedirá permissão da câmera

Como atualizar:
1. Suba estes arquivos para o GitHub
2. Render > Manual Deploy > Clear build cache & deploy
3. Abra /setup se necessário
4. Use normalmente\n
