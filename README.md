VERSÃO PWA - APP INSTALÁVEL NO CELULAR

O que foi incluído:
- manifest.webmanifest
- service-worker.js
- ícone do app com a logo
- abertura em modo standalone
- instalação na tela inicial

Como publicar:
1. Extraia este ZIP no computador
2. No GitHub, substitua:
   - app.py
   - pasta templates
   - pasta static
   - requirements.txt
3. No Render:
   - Manual Deploy
   - Clear build cache & deploy

Como instalar no celular:
Android / Chrome:
- Abra o sistema
- Toque nos 3 pontos
- Instalar app ou Adicionar à tela inicial

iPhone / Safari:
- Abra o sistema
- Compartilhar
- Adicionar à Tela de Início
